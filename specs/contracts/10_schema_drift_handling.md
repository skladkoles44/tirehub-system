# Schema Drift Handling

## Назначение
Данный файл определяет только классификацию schema drift.
Операционное решение по run задаётся в `06_quality_gates.md`.

## Drift classes

### optional_added
Добавлен новый необязательный столбец или поле.

### optional_missing
Пропал необязательный столбец или поле.

### required_missing
Пропал обязательный столбец или поле.

### type_changed
Изменился тип существующего поля.

### structure_changed
Изменилась структура JSON, вложенного объекта или табличной формы так, что прежний extract contract больше не применим.

## Правила
- Schema drift должен классифицироваться явно.
- Не должно быть состояния "молча проскочил".
- Один обнаруженный drift event получает ровно один drift class.
- Несколько изменений в одном run могут давать несколько drift events.

## Источник истины
- Этот файл отвечает только за drift classification.
- `06_quality_gates.md` отвечает только за gate action.
